GovQueue is a smart Queue & Token Management System built using the MERN stack.
It allows citizens to book tokens online, check live queue updates, and receive SMS notifications.
Staff can manage queues, call the next customer, and monitor crowd flow.
This project improves service efficiency, reduces waiting time, and enhances user experience in public service environments.
Features:
1)Online token booking
2)Live queue status (Socket.io)
3)Staff dashboard to call/skip tokens
4)Admin panel for service & counter management
5)Secure login (JWT)
6)Responsive UI

Tech Stack:-
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Real-time: Socket.io
